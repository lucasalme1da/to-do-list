## 💡 Funcionalidades 

- Colocar um nome personalizado, podendo alterá-lo quando desejar;
- Adicionar novas tarefas em formatos de cartões;
- Alterar os cartões em até 8 cores diferentes (cinza, azul, laranja, verde, vermelho, rosa, roxo ou amarelo);
- Ordernar as tarefas da maneira que quiser;
- Colocar as tarefas como finalizadas, e depois, se necessário, excluí-las; 
- Salvamento automático no navegador (obs: caso o cache do navegador for apagado, as tarefas também serão deletadas, bem como o nome de usuário).
